# RapidIdentity_spray
For Spraying RapidIdentity Portals
```
usage: spray.py [-h] -u USERS -p PASSWORDS -t TARGET

This is a tool to brute force RapidIdentity IAM Portal

optional arguments:
  -h, --help            show this help message and exit
  -u USERS, --users USERS
                        Input file name
  -p PASSWORDS, --passwords PASSWORDS
                        Wordlist file name
  -t TARGET, --target TARGET
                        target hostname
```
